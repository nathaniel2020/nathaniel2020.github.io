Windows下基于Python+Pytorch的深度学习环境的配置

使用必读

  1. 本机系统：win10
  2. 环境配置如果是从零开始按本教程配置即可，若是只配置（安装）单个阶段，请注意依赖关系
  3. 电脑里一般都有NVIDA的驱动，我是在安装CUDA或者CUDNN后，验证CUDNN安装失败，又发现在桌面右键NVIDA控制面板不见了，故而重新安装了NVIDA驱动，解决了问题。
  4. 若电脑中无NVIDA驱动，建议先安装驱动，这样方便安装对应版本的CUDA

配置python

  1. Windows store搜索python下载即可

安装CUDA

  1. 查看电脑的驱动程序版本

    1. 在桌面右键，打开NVDIA控制面板

    2. 点击左下角系统信息，查看驱动程序版本，如图。

  2. 选择合适CUDA版本，如下图，或见此

  3. 下载并安装，下载地址，网速还行就选network版吧

    1. 注：安装时选择自定义，并不勾选 visual studio integration，这是vs的一个扩展，网上很多人说安装整个话可能会失败，所有选择不安装保险一些，日后需要使用时再安装

  4. 验证安装是否成功

    1. 打开Terminal，输入nvcc -V,如图所示，即安装成功

CUDNN安装

  1. 下载文件，见此,需注册并填写问卷

  2. 解压，得到如下文件

  3. 将这些文件，复制到CUDA的安装目录(C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v9.2),版本号按自己的情况修改。

  4. 验证是否安装成功(两个都运行一下吧)

    1. 进入C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v9.2\extras文件夹，在此处打开Terminal

      1. 运行bandwidthTest.exe, result=pass即代表成功

      2. 运行deviceQuery.exe,result=pass即代表成功

NVDIA驱动安装

  1. 下载驱动程序,见此，选择合适的版本，注意产品系列一般都是Series，而不是notebook

  2. 下载完，安装即可。

  3. 验证，此时在桌面单击鼠标右键，应又NVDIA控制面板的选项，另进入C:\Program Files\NVIDIA Corporation\NVSMI文件夹，打开Terminal运行nvidia-smi.exe,如图所示即可。

安装Anaconda+修改powershell配置

  1. 下载

  2. 安装，建议勾选将Anaconda添加到环境变量

  3. 打开Terminal(powershell)，输入conda -V`,如图所示，即安装成功(不行的话，重启电脑就欧克了)

  4. 配置powershell（必须要配置，要不然不显示当前虚拟环境，以及无法激活虚拟环境）

    1. 更改powershell的执行策略

      1. 以管理员权限打开powershell，键入set-executionpolicy remotesigned,输入Y即可。

    2. 打开powershell，键入conda init powershell,再重启powershell即可

Pytorch安装

  1. 在官网选择合适的版本

  2. 打开Terminal，激活自己的conda环境，键入Run this Command所示的代码进行安装

  3. 验证安装是否成功

  4. 验证是否可以使用GPU进行训练

    1. 新建python文件，内容如下

               import torch
         import torch.nn as nn
         import torch.nn.functional as F
         import torch.optim as optim
         from torchvision import datasets, transforms
         ​
         BATCH_SIZE=512 #大概需要2G的显存
         EPOCHS=20 # 总共训练批次
         DEVICE = torch.device("cuda" if torch.cuda.is_available() else "cpu")
         ​
         train_loader = torch.utils.data.DataLoader(
                 datasets.MNIST('data', train=True, download=True, 
                                transform=transforms.Compose([
                                    transforms.ToTensor(),
                                    transforms.Normalize((0.1307,), (0.3081,))
                                ])),
                 batch_size=BATCH_SIZE, shuffle=True)
         ​
         test_loader = torch.utils.data.DataLoader(
                 datasets.MNIST('data', train=False, transform=transforms.Compose([
                                    transforms.ToTensor(),
                                    transforms.Normalize((0.1307,), (0.3081,))
                                ])),
                 batch_size=BATCH_SIZE, shuffle=True)
         ​
         class ConvNet(nn.Module):
             def __init__(self):
                 super().__init__()
                 # batch*1*28*28（每次会送入batch个样本，输入通道数1（黑白图像），图像分辨率是28x28）
                 # 下面的卷积层Conv2d的第一个参数指输入通道数，第二个参数指输出通道数，第三个参数指卷积核的大小
                 self.conv1 = nn.Conv2d(1, 10, 5) # 输入通道数1，输出通道数10，核的大小5
                 self.conv2 = nn.Conv2d(10, 20, 3) # 输入通道数10，输出通道数20，核的大小3
                 # 下面的全连接层Linear的第一个参数指输入通道数，第二个参数指输出通道数
                 self.fc1 = nn.Linear(20*10*10, 500) # 输入通道数是2000，输出通道数是500
                 self.fc2 = nn.Linear(500, 10) # 输入通道数是500，输出通道数是10，即10分类
             def forward(self,x):
                 in_size = x.size(0) # 在本例中in_size=512，也就是BATCH_SIZE的值。输入的x可以看成是512*1*28*28的张量。
                 out = self.conv1(x) # batch*1*28*28 -> batch*10*24*24（28x28的图像经过一次核为5x5的卷积，输出变为24x24）
                 out = F.relu(out) # batch*10*24*24（激活函数ReLU不改变形状））
                 out = F.max_pool2d(out, 2, 2) # batch*10*24*24 -> batch*10*12*12（2*2的池化层会减半）
                 out = self.conv2(out) # batch*10*12*12 -> batch*20*10*10（再卷积一次，核的大小是3）
                 out = F.relu(out) # batch*20*10*10
                 out = out.view(in_size, -1) # batch*20*10*10 -> batch*2000（out的第二维是-1，说明是自动推算，本例中第二维是20*10*10）
                 out = self.fc1(out) # batch*2000 -> batch*500
                 out = F.relu(out) # batch*500
                 out = self.fc2(out) # batch*500 -> batch*10
                 out = F.log_softmax(out, dim=1) # 计算log(softmax(x))
                 return out
         ​
         ​
         model = ConvNet().to(DEVICE)
         optimizer = optim.Adam(model.parameters())
         ​
         def train(model, device, train_loader, optimizer, epoch):
             model.train()
             for batch_idx, (data, target) in enumerate(train_loader):
                 data, target = data.to(device), target.to(device)
                 optimizer.zero_grad()
                 output = model(data)
                 loss = F.nll_loss(output, target)
                 loss.backward()
                 optimizer.step()
                 if(batch_idx+1)%30 == 0: 
                     print('Train Epoch: {} [{}/{} ({:.0f}%)]\tLoss: {:.6f}'.format(
                         epoch, batch_idx * len(data), len(train_loader.dataset),
                         100. * batch_idx / len(train_loader), loss.item()))
         ​
         ​
         def test(model, device, test_loader):
             model.eval()
             test_loss = 0
             correct = 0
             with torch.no_grad():
                 for data, target in test_loader:
                     data, target = data.to(device), target.to(device)
                     output = model(data)
                     test_loss += F.nll_loss(output, target, reduction='sum').item() # 将一批的损失相加
                     pred = output.max(1, keepdim=True)[1] # 找到概率最大的下标
                     correct += pred.eq(target.view_as(pred)).sum().item()
         ​
             test_loss /= len(test_loader.dataset)
             print('\nTest set: Average loss: {:.4f}, Accuracy: {}/{} ({:.0f}%)\n'.format(
                 test_loss, correct, len(test_loader.dataset),
                 100. * correct / len(test_loader.dataset)))
         ​
         for epoch in range(1, EPOCHS + 1):
             train(model, DEVICE, train_loader, optimizer, epoch)
             test(model, DEVICE, test_loader)

    2. 在装有pytorch的环境下运行该文件，当其到达train阶段时（如图1），在C:\Program Files\NVIDIA Corporation\NVSMI文件夹，打开Terminal运行nvidia-smi.exe,如图2所示即成功。
